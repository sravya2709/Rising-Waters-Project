# 🌊 Rising Waters – A Flood Prediction

---

# 📖 Project Overview

**Rising Waters** is an AI-powered Flood Prediction Web Application that predicts the likelihood of flood occurrence using historical weather and rainfall data. The system leverages Machine Learning algorithms to analyze meteorological parameters and provide early flood risk predictions through an interactive web interface.

The primary objective of this project is to support disaster preparedness by providing timely flood predictions based on environmental conditions.

---

# ✨ Features

- 🌧️ Predict flood occurrence using weather parameters
- 🤖 Machine Learning-based prediction system
- 🌐 Interactive Flask web application
- 📊 Data preprocessing and visualization
- ⚖️ Multiple Machine Learning models for comparison
- 🚀 XGBoost selected as the final prediction model
- ⚡ Instant prediction results
- 💻 User-friendly and responsive interface

# 🤖 Machine Learning Models

The following algorithms were implemented and evaluated:

- 🌳 Decision Tree
- 🌲 Random Forest
- 🤝 K-Nearest Neighbors (KNN)
- 🚀 XGBoost

After comparing the performance of all models, **XGBoost** was selected as the final deployment model due to its excellent predictive performance and generalization capability.

# ▶️ How To Run

Clone the repository:

```bash
git clone https://github.com/sravya2709/Rising-Waters-Project.git
```

Navigate to the project folder:

```bash
cd Rising-Waters-Project
```

Install the required libraries:

```bash
pip install -r requirements.txt
```
Run the Flask application:

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```
